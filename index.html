How it works
Three moving parts, fully automated.
01

OBS runs the replay buffer

OBS Studio continuously holds the last 30 seconds of your screen in memory. Lightweight — no disk usage until a clip is saved.

02

Work Watch monitors your active window

A small Python script polls your active window every few seconds. When it detects Unity, Blender, or Substance Painter, it starts tracking your work session.

03

Clips save automatically

After 1 minute of focused work, it tells OBS to save the replay buffer — then again every 15 minutes. You get a toast notification each time. Clips land in a folder ready for social media or archiving your projects progress.

Features
Built for devs who forget to hit record.
👁️

Smart Detection

Monitors your active window title. Only saves clips when you're actually working in a target app — not when you're browsing Reddit.

⏱️

Focus Check

Waits for 1 minute of continuous focus before saving the first clip. Brief alt-tabs through an app won't trigger a save.

🔔

Toast Notifications

Get a Windows notification each time a clip is saved so you always know it's working without checking the terminal.

🚀

Starts on Boot

Configure OBS and WorkWatch to launch on startup. Your entire WIP capture pipeline runs without you lifting a finger.

⚙️

Configurable

Change target apps, clip interval, focus time, and OBS connection settings by editing a few lines at the top of the script.

🪶

Lightweight

A single Python script under 300 lines. No framework, no overhead. Runs alongside OBS with negligible resource usage.

Quick Start

Up and running in 5 minutes.

0. Download WorkWatch.py
Download the file from GitHub
1. Install dependencies
pip install obsws-python pywin32 plyer
2. Configure OBS Studio
# Enable WebSocket Server
Tools → WebSocket Server Settings → Enable

# Enable Replay Buffer (30 seconds)
Settings → Output → Replay Buffer → Enable

# Set recording format and path
Settings → Output → Recording → MP4, choose a folder

# Add a source and start the buffer
Add Display/Window Capture → Click "Start Replay Buffer"
3. Run WorkWatch
python WorkWatch.py
